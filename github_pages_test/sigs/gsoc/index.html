Google Summer of Code

Overview

This special interest group organizes Google Summer of Code in the Jenkins organization. Scope of work includes coordinating GSoC and working with potentials mentors and students in order to form GSoC project teams.

Description

This SIG focuses on Google Summer of Code and other similar event. The group drives the GSoC subproject in the Jenkins organization. The scope of work includes organizing the events and working with potentials mentors and students in order to organize GSoC teams.

You can find more info about GSoC in Jenkins here.

Meetings

  • GSoC SIG meetings happen monthly during the entire year

  • See the GSoC subproject for the list of public events within Google Summer of Code timeframe

  • Other meetings will be announced in GSoC mailing lists
